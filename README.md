# sesam
Use Sesam to create easy collapsing items on your site

## Install
### With NPM

**1. Install package**

`npm i sesam-collapse`

**2. Import package**

```javascript
import {sesam} from 'sesam-collapse';
```

### With a CDN
Just copy paste this link-tag in your site's head
```html
<link rel="stylesheet" href="">
```
